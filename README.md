# Task-2

1.int x = -7; dəyişənini təyin edin. Proqram ekranda "Mənfi", "Müsbət" və ya "Sıfır" yazsın.(if-else)

2.int a = 15, b = 28; — bu iki ədəddən böyüyünü ekrana çıxarın. Bərabər olduqda "Bərabərdir" yazsın.(if-else)

3.int n = 12; — ədədin həm cüt/tək, həm də müsbət/mənfi olduğunu ayrıca ekranda göstərin. (İç-içə if istifadə edin.)(if-else)

4.int x = 10, y = 4; char op = '+'; — op dəyişəninə görə toplama, çıxma, vurma və bölmə əməliyyatı aparsın. Sıfıra bölmə halında xəbərdarlıq çıxarsın.(swich-case)

5.int ay = 11; — ay nömrəsinə görə fəsli ("Yaz", "Yay", "Payız", "Qış") ekrana çıxarın. Birdən çox case eyni nəticəyə aparmalıdır.(swich-case)
